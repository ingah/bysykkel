## Hvordan starte applikasjonen

#### Unix, Linux eller Mac
```
> git clone https://github.com/ingah/bysykkel.git
> ./mvnw install
> java -jar target/bysykkel-0.0.1-SNAPSHOT.jar
```

Åpne en nettleser og tast inn http://localhost:8080/bysykkel

#### Windows
```
> git clone https://github.com/ingah/bysykkel.git
> mvnw.cmd install
> java -jar target\bysykkel-0.0.1-SNAPSHOT.jar
```

Åpne en nettleser og tast inn http://localhost:8080/bysykkel
